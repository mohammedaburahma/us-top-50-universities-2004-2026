# us-top-50-universities-2004-2026

This project explores a dataset containing information about the top 50 universities in the United States, including characteristics such as ranking positions, enrollment figures, tuition costs, acceptance rates, and more. The analysis aims to answer questions like:

Which universities have the highest rankings?

How do metrics such as enrollment and tuition vary across institutions?

Are there any visible trends or correlations among numerical variables?
